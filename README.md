# 接口文档转模型参数工具


mac开发小工具

用于将后台给的接口文档直接转换成模型里的property属性，并保留注释

### 使用方法1：
1. 将项目的开发者账号修改成你个人的账号（免费的就可以）
2. 将项目运行起来-目标选择当前mac电脑
3. 在Dock中右键app图标-选项-图标-在Dock中保留
4. app中间的图片为转换按钮

### 使用方法2：
如果不想自己对代码进行自定义修改的话
也可以直接将项目中的app文件下的文件直接复制到/Applications目录下
并在系统设置-安全性与隐私-中允许允许即可
